# Tripadvisor-clone

This project is a simple clone of the TripAdvisor website, created using HTML and CSS. The goal is to replicate the basic layout and styling of the TripAdvisor homepage.

*Features*

Responsive layout for different devices
Section for Award Winning Advendure places
Section for ways to tour chennai
Section for liked places
Section for top places to visit
Section for exploring new places
Footer with links and information
